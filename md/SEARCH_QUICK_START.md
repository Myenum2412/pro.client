## Troubleshooting

### Q: Search doesn't open?
**A:** Try clicking the "Search files..." button instead

**A:** Try:
- Different keywords
- Partial name (e.g., "rep" instead of "report")
- File extension (e.g., ".pdf")

### Q: Can't see recent searches?
**A:** They appear only when search box is empty

### Q: Keyboard shortcut not working?
**A:** Check if another app is using Ctrl+K
